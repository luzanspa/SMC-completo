1. 🏥 Introdução ao SCEL

O Sistema de Cadastro de Exames Laboratoriais (SCEL) é uma ferramenta completa para gestão de pacientes e exames médicos. Desenvolvido para ser intuitivo e eficiente, oferece recursos avançados de cadastro, busca, estatísticas e backup.

🎯 Principais Funcionalidades:
  • Cadastro completo de pacientes com validações
  • Seleção de exames por categorias
  • Busca avançada em tempo real
  • Dashboard com estatísticas detalhadas
  • Sistema de backup automático
  • Histórico inteligente de CNS

2. 📋 Cadastro de Pacientes

Campos Obrigatórios:

  • Nome Completo: Mínimo 2 caracteres, convertido automaticamente para maiúsculas
  • CNS/CPF: Aceita CNS (15 dígitos) ou CPF (11 dígitos) com formatação automática
  • Data de Nascimento: Validada para idades entre 0-120 anos
  • Gênero: Opções pré-definidas ou campo livre para "Outro"
  • Laboratório: Apolo, Queiroz ou São Lourenço
  • Endereço: Mínimo 10 caracteres
  • Exames: Pelo menos um exame deve ser selecionado
💡 Dica CNS: O sistema prioriza o formato CNS. Digite mais de 11 dígitos para formatação automática como CNS (000 0000 0000 0000).

Histórico Inteligente:

Ao digitar 3 ou mais dígitos no campo CNS/CPF, o sistema exibe automaticamente um histórico de pacientes similares. Use as setas ↑↓ para navegar e Enter para selecionar.

3. 🔍 Sistema de Busca

A busca é realizada em tempo real em todos os campos do paciente:

  • Nome completo
  • CNS/CPF (com ou sem formatação)
  • Data de nascimento
  • Telefone
  • Gênero
  • Laboratório
  • Endereço
  • Exames selecionados
  • Data de cadastro
⚡ Performance: A busca usa debounce de 300ms e cache para otimizar a performance mesmo com muitos registros.

4. 🧪 Seleção de Exames

O sistema possui um catálogo completo de exames organizados por categorias:

Categorias Disponíveis:

  • Hematologia: Hemograma, VHS, Grupo sanguíneo, etc.
  • Bioquímica: Glicose, Colesterol, Creatinina, etc.
  • Endocrinologia: TSH, T3, T4, HCG, etc.
  • Sorologia: HIV, Hepatites, Toxoplasmose, etc.
  • Urina e Fezes: EAS, Urocultura, Parasitológico, etc.
  • Coagulação: TP, TTPA, etc.

Como Selecionar:

  1. Clique em "Selecionar Exames"
  2. Navegue pelas categorias organizadas em colunas
  3. Clique nos exames desejados (ficarão destacados em verde)
  4. Confirme a seleção

5. ✏️ Edição e Exclusão

Editando Pacientes:

  1. Clique no botão ✏️ na linha do paciente
  2. Os dados serão carregados no formulário principal
  3. Faça as alterações necessárias
  4. Clique em "Salvar" ou use Ctrl+S

Exclusão Segura:

O sistema possui tripla confirmação para exclusões:

  1. Primeira confirmação com dados do paciente
  2. Digitação do nome completo para confirmar
  3. Confirmação final antes da exclusão
⚠️ Atenção: A exclusão é permanente e não pode ser desfeita. Mantenha backups regulares.

6. 📊 Dashboard e Estatísticas

O dashboard oferece análises completas dos dados cadastrados:

Métricas Principais:

  • Total de Pacientes: Contador geral com percentual de crescimento
  • Cadastros Hoje: Registros do dia atual
  • Cadastros do Mês: Com média diária
  • Taxa de Contato: Percentual de pacientes com telefone
  • Valores Financeiros: Gastos totais e média por paciente

Análises Disponíveis:

  • Distribuição por gênero e faixa etária
  • Gastos por laboratório e categoria
  • Top 15 exames mais solicitados
  • Pacientes mais recorrentes
  • Cadastros mensais (gráfico)
  • Qualidade dos dados de contato
📅 Filtros: Use o seletor de mês para analisar períodos específicos. O dashboard pode ser impresso para relatórios.

7. 💾 Backup e Importação

Backup Automático:

O sistema cria backups automaticamente:

  • A cada cadastro, edição ou exclusão
  • Arquivo Excel com timestamp
  • Dados ordenados por data (mais recentes primeiro)
  • Lembrete diário para backup manual

Exportação Manual:

  1. Clique em "Exportar Excel" ou use Ctrl+E
  2. Arquivo será baixado com nome: SCEL_dados_AAAA-MM-DD_HHhMM.xlsx
  3. Contém todos os dados formatados para planilha

Importação:

⚠️ CUIDADO: A importação substitui TODOS os dados existentes. É uma ação irreversível com tripla confirmação.
  1. Clique em "Importar Excel"
  2. Selecione arquivo .xlsx ou .xls
  3. Confirme as três etapas de segurança
  4. Aguarde o processamento
